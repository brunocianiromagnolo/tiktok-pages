Política de Privacidad

Última actualización: 19 de mayo de 2025

ReelPilot es una plataforma de programación y publicación automática de contenido en redes sociales.

1. Información que recopilamos

  • Correo electrónico (para identificar tu cuenta)
  • Tokens OAuth de YouTube, TikTok, Instagram y Facebook (para publicar en tu nombre)
  • Videos y metadatos que subís para programar

2. Cómo usamos tu información

  • Publicar en YouTube Shorts, TikTok, Instagram Reels y Facebook según tus instrucciones
  • Gestionar tu cuenta y programación de contenido

No vendemos ni compartimos tu información con terceros.

3. Uso de la API de YouTube

ReelPilot usa la API de YouTube para publicar videos. Al conectar tu cuenta aceptás las Condiciones del Servicio de YouTube.

Podés revocar el acceso desde myaccount.google.com/permissions.

Los datos obtenidos vía API de YouTube están sujetos a la Política de Privacidad de Google.

4. Uso de la API de TikTok

Usamos la API de TikTok para publicar videos. Al conectar tu cuenta aceptás las Condiciones de TikTok y su Política de Privacidad.

5. Uso de la API de Facebook e Instagram

Usamos la API de Meta para publicar en Facebook e Instagram. Al conectar tu cuenta aceptás las Políticas de Meta.

6. Tus derechos

  • Eliminar tu cuenta y datos en cualquier momento
  • Revocar acceso a YouTube: myaccount.google.com/permissions
  • Revocar acceso a TikTok/Facebook/Instagram desde sus configuraciones
